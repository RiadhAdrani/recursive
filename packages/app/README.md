## `RecursiveApp`

Create an App instance.

Automatically initialize `Orchestrator`, `State`, `Renderer` and `Router`.

Exposed methods:

-   _`render`_
-   _`route`_
-   _`getRoute`_
-   _`getParams`_
-   _`getBase`_
-   _`goTo`_
-   _`renderRoute`_
-   _`getState`_
-   _`setState`_
-   _`getCache`_
-   _`setCache`_
-   _`getRef`_
-   _`updateOn`_
